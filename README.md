# React-game: Sapper

Для проверки приложения можно воспользоваться готовым деплоем - [netlify](https://dansitnikov-react-game-sapper.netlify.app/aboutGame)

## Downloading

```
git clone {repository URL}
```

## Installing NPM modules

```
npm install
```

## Running application

```
npm run start
```

## Checking eslint

```
npm run lint
```
